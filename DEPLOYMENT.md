# 🌍 GeoChallenge Game - Quick Deployment Guide

## 🚀 One-Command Installation

### Windows (PowerShell)
```powershell
wget.exe --show-progress -O install.bat https://raw.githubusercontent.com/testgk/pandapandas/master/deploy/install_windows.bat ; .\install.bat
```

### Linux/macOS
```bash
wget --show-progress -O install.sh https://raw.githubusercontent.com/testgk/pandapandas/master/deploy/install_linux.sh ; bash install.sh
```

## 🎮 After Installation

### **Launch the Game**
```bash
# Windows
run_geochallenge.bat

# Linux/macOS  
./run_geochallenge.sh
```

### **Game Modes Available**
1. **🌍 3D Interactive Globe** - Recommended, click on Earth
2. **📝 Text Coordinate Input** - Enter lat/lon manually  
3. **🌐 Web-based Map** - Click on interactive web map

### **Sample Challenges by Difficulty**
- **Easy (25 cities)**: New York, London, Paris, Tokyo, Sydney
- **Medium (44 cities)**: Barcelona, Istanbul, Prague, Mumbai
- **Hard (32 cities)**: Ulaanbaatar, Reykjavik, Antananarivo  
- **Expert (34 cities)**: Funafuti, Ngerulmud, Nuuk, São Tomé
