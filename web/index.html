    🌍 How to Play

    Objective

    Find locations on the globe! Click where you think the city/landmark is located.

    Scoring

    • 🟢 Green Zone (0-50km): 80-100 points
    • 🟡 Yellow Zone (50-150km): 50-80 points
    • 🟠 Orange Zone (150-300km): 20-50 points
    • 🔴 Red Zone (300-500km): 1-20 points
    • ❌ Outside Country: -80% penalty

    Hint Penalties

    • Country (-20%): Reveals the country name
    • Hint (-10%): Shows a text hint (can use multiple)
    • Zoom In! (-50%): Camera zooms near the target

    Tips

    • Use the continent info shown at the start
    • Hints stack - using all hints = -80% total
    • Build streaks for bonus points!
